### 各种插件和集成的集合

每个插件都有自己的文件夹,均为maven module, 请按需获取.

```xml
		<dependency>
			<groupId>org.nutz</groupId>
			<artifactId>填nutz插件名</artifactId>
			<version>1.r.67</version>
		</dependency>
```

# 快照版地址

http://8go.win:4508/repository/nutzcn-snapshots//org/nutz/

```xml
	<repositories>
		<repository>
			<id>nutzcn-snapshots</id>
			<url>http://8go.win:4508/repository/nutzcn-snapshots/</url>
			<snapshots>
				<enabled>true</enabled>
			</snapshots>
		</repository>
	</repositories>
	<dependencies>
		<dependency>
			<groupId>org.nutz</groupId>
			<artifactId>填nutz插件名</artifactId>
			<version>填版本号-SNAPSHOT</version>
		</dependency>
		<!-- 其他依赖 -->
	</dependencies>
```
